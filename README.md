# docker-latex

The `dockerfiles` to build [csorger/latex](https://hub.docker.com/r/csorger/latex)
on [hub.docker.com](https://hub.docker.com).

The images contain a distribution of latex and friends:

- texlive-full
- biber
- latexmk
- pandoc
- pandoc-citeproc

on various Debian versions:

## Versions

- TeXlive 2014.20141024 : jessie (oldoldstable)
- TeXlive 2016.20170123 : stretch (oldstable)
- TeXlive 2018.20190227 : buster (stable)
- TeXlive 2020.20200417 : bullseye (testing)
- TeXlive 2020.20200417 : sid (unstable)
